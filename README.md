# Stylish - Fashion Shopping App

## About Stylish

Stylish is a modern fashion shopping mobile application built with Flutter.
It provides a smooth shopping experience for browsing clothes, shoes,
accessories, and beauty products through a clean UI and fast performance.

The app is designed for real e-commerce use, with product categories, product
details, cart functionality, wishlist management, user authentication, checkout,
and customer order tracking.

## Features

- Browse fashion categories
- Search and filter products
- Wishlist and favorites
- Add to cart
- Checkout with Supabase order records
- Track order status
- Product details page
- User authentication
- Supabase backend integration

## Tech Stack

- Framework: Flutter
- Language: Dart
- Backend: Supabase
- Database: Supabase Database
- Authentication: Supabase Auth
- State Management: Bloc
- Architecture: Clean Architecture
