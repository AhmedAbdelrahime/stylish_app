📱 Stylesh – Fashion Shopping App
🧵 About Stylesh

Stylesh is a modern fashion shopping mobile application built with Flutter.
It provides a smooth and elegant shopping experience for browsing clothing, shoes, and accessories with a clean UI and fast performance.

The app is designed for scalability and real-world e-commerce use, supporting product categories, product details, and user interactions.

✨ Features

🛍️ Browse fashion categories (Clothes, Shoes, Accessories, Beauty)

🔎 Search and filter products

❤️ Wishlist / favorites

🛒 Add to cart

📦 Product details page

👤 User authentication

🔥 Modern UI design

⚡ Fast performance with Flutter

☁️ Firebase backend integration

🏗️ Tech Stack

Framework: Flutter

Language: Dart

Backend: supabase

Database: Cloud subapase

Auth: Firebase Authentication

State Management: Bloc 

Architecture: Clean Architecture
