# 📱 Stylish – Fashion Shopping App

## 🧵 About Stylish
Stylish is a modern fashion shopping mobile application built with Flutter.  
It provides a smooth and elegant shopping experience for browsing clothes, shoes, accessories, and beauty products through a clean UI and fast performance.

The app is designed for scalability and real-world e-commerce use, supporting product categories, product details, cart functionality, wishlist management, and user authentication.

## ✨ Features
- 🛍️ Browse fashion categories (Clothes, Shoes, Accessories, Beauty)
- 🔎 Search and filter products
- ❤️ Wishlist / favorites
- 🛒 Add to cart
- 📦 Product details page
- 👤 User authentication
- 🔥 Modern UI design
- ⚡ Fast performance with Flutter
- ☁️ Supabase backend integration

## 🏗️ Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **Backend:** Supabase
- **Database:** Supabase Database
- **Authentication:** Firebase Authentication
- **State Management:** Bloc
